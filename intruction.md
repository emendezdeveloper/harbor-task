# Microservices Security Audit Task

You are a backend engineer tasked with auditing a distributed microservices system for security vulnerabilities.

The system consists of three services:

* Authentication Service
* Orders Service
* Payments Service

Each service may contain security vulnerabilities such as:

* Improper authentication validation
* Missing authorization checks
* Insecure data handling

## Your task:

1. Identify all security vulnerabilities across the services
2. Provide the file name and line number
3. Explain why it is a vulnerability

## Constraints:

* You must analyze all services
* Partial results will be considered incorrect
